## `>_ whoami`

Lum0s Solutions is a security-first engineering lab building **Syn_OS** ‚ÄĒ an AI-enhanced cybersecurity operating system built from the ground up on **Arch Linux**. We operate at the intersection of kernel engineering, autonomous AI agents, distributed mesh computing, and offensive security research.

This isn't a reskin. It's a **122-crate Rust workspace** with a **custom Linux 6.19-synos-ai kernel** (11 custom syscalls, 12 Rust kernel modules), **960+ source files**, and AI-native tooling woven into every layer ‚ÄĒ from post-quantum cryptography to gamified SOC training. v26-v30 delivered in a single codesprint.
